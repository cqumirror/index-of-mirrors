关于此镜像站

重庆大学开源镜像站由重庆大学蓝盟维护管理。目前收录了CentOS、Archlinux、CentOS、Ubuntu等多个发行版的镜像。依托重庆大学网络面向全国提供优质的镜像源服务。

镜像源说明

1. 本镜像站支持IPv4访问
2. 本镜像站支持HTTP协议访问
3. 本镜像站更新频率为一天二次

下载说明

请勿使用多线程下载工具。公共资源，带宽有限，请使用浏览器自带下载工具下载安装包，谢谢！ Why? Public sources, bandwith Limited. Use default download tools only, Please.
